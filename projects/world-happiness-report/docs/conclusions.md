# World Happiness Report 分析结论

## 1. 高幸福分数国家集中在西欧、北美和澳新地区

发现：2019 年幸福分数最高的是 Finland，分数为 7.769；Denmark 和 Norway 分别排第 2、3 名。地区均值上，Australia and New Zealand 为 7.27，North America 为 7.09，Western Europe 为 6.90，整体明显高于其他地区。

可能原因：这些地区在人均 GDP、社会支持、健康预期寿命等指标上整体较高。以 Western Europe 为例，2019 年平均 GDP 指标约为 1.36，平均社会支持约为 1.48，平均健康预期寿命约为 1.01。

建议：如果把这份数据用于海外市场环境初筛，西欧、北美和澳新可以作为优先观察区域。但这只能作为宏观环境参考，不能单独决定市场进入策略。

## 2. 低幸福分数国家集中在撒哈拉以南非洲和部分高风险地区

发现：2019 年最低分国家是 South Sudan，分数为 2.853；Central African Republic 和 Afghanistan 也处于底部。Sub-Saharan Africa 的 2019 年平均分约为 4.29，是主要地区中最低的一组。

可能原因：这些国家在人均 GDP、社会支持、健康预期寿命等指标上整体偏低。例如 Sub-Saharan Africa 2019 年平均 GDP 指标约为 0.45，平均社会支持约为 0.92，平均健康预期寿命约为 0.41。

建议：如果用于商业选址或国际业务分析，这类地区需要优先补充政治风险、基础设施、消费能力和安全环境数据，不能只用幸福指数下判断。

## 3. 2015-2019 年国家变化差异很大

发现：在 2015 和 2019 都出现的国家中，Benin 幸福分数上升最多，从 3.340 上升到 4.883，变化为 +1.543；Venezuela 下降最多，从 6.810 下降到 4.707，变化为 -2.103。

可能原因：单个国家的变化可能与经济环境、社会稳定性、公共服务、外部冲突等因素有关。但当前数据只有年度聚合指标，不能直接识别具体原因。

建议：变化分析适合作为预警线索。对上升或下降明显的国家，应进一步结合宏观经济、政治风险、人口结构和政策变化数据做解释。

## 4. GDP、健康预期寿命和社会支持与幸福分数关系最强

发现：全量 2015-2019 数据中，GDP 与幸福分数相关系数约为 0.789，健康预期寿命约为 0.742，社会支持约为 0.649。自由度相关性中等，腐败感知和慷慨程度相关性相对较弱。

可能原因：经济水平、健康条件和社会支持更接近居民生活质量的基础条件，因此与幸福分数同步变化更明显。

建议：报告表达时应写成“相关性较强”，不能写成“GDP 导致幸福分数提高”。如果要做因果判断，需要加入更长时间序列、更多控制变量和严谨模型。

## 5. 这个项目更适合展示基础分析能力，不适合作为预测建模项目

发现：项目已经覆盖字段统一、缺失值检查、跨年合并、Top/Bottom、变化分析、地区对比、相关性分析和可视化表达。合并后数据为 782 行、11 列，覆盖 2015-2019 年。

可能原因：数据是国家级聚合数据，样本量有限，且部分解释变量本身就是幸福分数的组成维度。如果直接建模，容易变成用组成项预测总分。

建议：面试中应把这个项目定位为数据清洗和分析表达项目，重点讲清楚字段口径统一、共同国家比较、相关性边界和图表层不重新清洗原始数据的分层设计。
