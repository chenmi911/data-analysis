# Superstore 经营分析结论

## 1. 整体经营

- 数据共 9994 行，订单日期覆盖 2014-01-03 到 2017-12-30。
- 总销售额 2,297,200.86，总利润 286,397.02，整体利润率 12.47%。
- 2017 年销售额最高，为 733,215.26；2017 年利润也最高，为 93,439.27。

结论：整体销售规模在扩大，但利润率没有同步明显提升，不能只看销售额判断经营质量。

## 2. 主要利润问题

- 区域层面，Central 利润率最低，为 7.92%。
- 大类层面，Furniture 利润率最低，为 2.49%。
- 子品类层面，Tables 总利润为 -17,725.48，Bookcases 为 -3,472.56，Supplies 为 -1,189.10。

结论：利润问题集中在部分子品类，不是全盘亏损。后续应优先看 Tables、Bookcases 的定价、折扣和履约成本。

## 3. 折扣风险

- 无折扣订单利润率为 29.51%。
- 折扣超过 30% 的订单销售额为 259,543.49，但利润为 -125,006.78，利润率为 -48.16%，亏损率为 97.77%。
- 高折扣下亏损最严重的子品类是 Binders，利润为 -38,510.50，利润率为 -106.56%。

结论：高折扣是最明确的利润风险。建议对 `discount > 0.3` 的订单设置审批或折扣上限，尤其关注 Binders、Machines、Tables、Bookcases。

## 4. 客户维度与 RFM 分层

SQL 已补充客户利润贡献、高销售亏损客户、RFM 明细和 RFM 分层汇总。

- 利润贡献最高的客户是 Tamara Chand，销售额 19,052.22，利润 8,981.32，利润率 47.14%。
- 销售额最高的客户是 Sean Miller，销售额 25,043.05，但利润为 -1,980.74，利润率 -7.91%。
- 高销售但亏损客户中，Cindy Stewart 利润为 -6,626.39，Grant Thornton 利润为 -4,108.66，Sean Braxton 利润为 -2,082.75。
- RFM 使用 `recency_days`、`frequency`、`monetary` 分别打 1-5 分，并将客户分为高价值客户、重点发展客户、流失风险客户、高消费低利润客户等类型。

结论：客户分层不能只看消费金额。高消费客户不一定是高价值客户，必须同时看利润、折扣和购买频次。

## 5. 业务建议

1. 对 Tables、Bookcases、Supplies 做专项利润复盘。
2. 对 `discount > 0.3` 建立审批规则。
3. 客户运营不要只追高销售额客户，应同时看利润贡献。
4. 后续补充成本、物流、库存和营销数据，否则不能分析真实毛利、库存周转和广告 ROI。
